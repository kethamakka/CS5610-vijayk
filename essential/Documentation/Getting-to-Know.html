Getting to know the project


My project "YourSpace" is a content management system. It provides various useful widgets in a single webpage. With growing technologies, the customers are demanding the webpages to be more like desktop applications. This behavior has been tried to achieve in this project.This project has been deeloped as a part of CS-5610 - Web Development course in Spring 2015, at Northeastern University.

The basic operations in my project are

  • Adding a widget
  • Asking the widget for some data
  • Saving the data of liking
  • Retrieving The liked data

Widget Description

Type of widget Description
Google Googles the query and opens the resuls in a new browser tab
Weather Takes a location as input query and gets the current weather condition along with a forecase for the next five days from the Yahoo Weather API
Youtube Takes a query as input and returns a playlist of 20 mathced results embedded in a 4:3 player from Youtube API
Geolocator Takess a location as input and givess that location marked in a map. This widget is powered by Google Maps API. The map can be zoomed and moved around the marked location
Director Takes two locations and gives out the route between them. This widget also uses the https://developers.google.com/maps/.
What does it M. E. A. N The pun intended title describes a dictionary taken from Wordnik Dictionary API

Technologies used

HTML 5 All the pages in the project are developed using HTML 5
CSS 3 The major part of project styling is dne using Cascading Style Sheets 3
Bootstrap Enhance the look and feel of the web page by adding handsome containers and fonts.
jQuery Used in the middle tier for a significant part
jQuery UI The frontend animations like drags and resizing of tiles are performed with jQueryUI
Angular JS Used in rendering complex structures (rendering tables) on html pages and to handle some calls to webservices.
Node JS The server programming is done on Node JS
Express JS To give the node interpretor the ability to behave as a server
Mongo DB The database system of the project
Mongoose To establish a connection with MongoDB on a host environment like Openshift
Passport JS To authenticate the the user whie logging in and help with session while logging out
