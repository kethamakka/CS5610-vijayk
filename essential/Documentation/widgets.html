The Widgets

These are the hearts of the project. Each widges has a unique capabiity and each widget is described as follows.

Googles a query and opens it in a new window

Takes a query and gets weather from yahoo weather

Takes a query and gets a youtube playlist matching that query

Takes in a query and loads a map with the location marked

Takes two location queries and displays a route between the two places

Takes in a word and fetches meaning of it
