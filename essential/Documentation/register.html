Registration Page

This page will take the user details and creates an account for the user in the data base. An AJAX call to the server for a new "POST" request creates a place for the user. From there The user is taken to the login page.

The image shows the layout of the questions each user is asked. There are validations for each field. No field can be left empty and. The password and the confirmation password must be same. The password must between 8 and 12 characters and must contain

  • one Uppercase letter
  • one lowercase letter
  • one digit letter

When every requirement is met with, the user is taken to the log in page
