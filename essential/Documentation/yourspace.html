YourSpace Page


This page is the playground of the user. The icon will add a tile to the page. The will show the favorites of a person. The will display the user profile. The will display all the users registered to YourSpace .The icon will log out a person. To view favories, view the profile and to find friends, a person has to be logged in. So if an unregistered user tries to get the favorites, instead of giving the details, an alert box is shown and the user will be automatically redircted to the login page.
