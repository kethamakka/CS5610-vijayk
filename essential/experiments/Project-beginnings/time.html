About the experiment

This experiment is priliminary layer of the project being developed. It runs an hour clock on the webpage. The Clock is responsive. The area holding the date and time is draggable. Double click on the time, changes the hour pattern of the clock (12 hours to 24 hours and vice versa). The double click change was achieved using the angular directives "ng-show", "ng-hide" and "ng-model". To get the date, Date.now() function has been used.

ng-model="am" ng-init="am = 0" ng-hide="am" ng-dblclick="changeType()
ng-show="am" ng-dblclick="changeTf()

The Code snippet shows how angular directives are used to achieve the functionality.

$scope.clock = Date.now() // get the current time
$timeout(tick, 1000); // Executes the function "tick" every 1000 milli seconds.
