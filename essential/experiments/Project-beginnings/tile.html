About the experiment

This experiment is about creating a tile. A tile is handler which holds the content being added to the tile. The widget that is being added will be given by the Dropdown menu on the tile header. The selected category will be applied to the tile. The tile is draggable. The user can customize the space available to him on the dashboard.
