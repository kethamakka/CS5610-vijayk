About the experiment

The experiment is one of the key aspects of the project. The "plus" glyphicon in the "navigation bar" will allow users to add a tile to the dashboard. The tile is added at an absolute position but the tile is draggable thus making the dashboard customizable.

Code snippet

$(".container").append(tile);
tile
  .draggable()
  .resizable()
  .css({
    "position": "absolute",
    "left" : x,
    "top" : y})
