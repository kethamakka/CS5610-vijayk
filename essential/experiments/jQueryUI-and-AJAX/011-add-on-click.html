About the Experiment

var p = $("<p contenteditable='true'>"); $("body").append(p);

This AJAX experiment was developed as an idea to mimic a content management system. The UI in the webpage allows the user to add a new paragraph. These new paragraphs can be moved around and resized. Thus beginning the idea of a content management system. The code snippet shows the java scipt used to achieve the functionality.
