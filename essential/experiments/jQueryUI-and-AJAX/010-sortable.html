The experiment is about the feature "sortable" of jQueryUI. This feature allows the lists to be sorted dynamically. The list items can be moved around in x or y axis and can be dropped iin the right place, while the rest os the list accomodate the changes by giving place. This idea was used to create a mini game to set right the image of Mona Lisa. The image was clipped using photoshop and the clipped sub images were used as list items and the list was made sortable. The above code snippet shows the procedure. In the CSS file, The list was displayed as "inline-block" style and the white spaces around the list elements were removed by setting the margins to negative values.
