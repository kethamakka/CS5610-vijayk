About the Experiment

The experiment is to show the fixed header on a web page. The header stays fixed in its position while the rest of the content scrolls along the page

Code Snippet

body header {
position:fixed;
width:920px;
}
The code show that the header's position is fixed and its width runs through 920px. The margins and paddings of the rest of the content has to be set according to the position of header.
