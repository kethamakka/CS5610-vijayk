About the Experiment

The Experiment is about responsive dessign. The css of the webpage uses the "@media" tag which refers to the screen of the display device. If the display device has a width grater then 700px, the body of the page will have cornflower blue and the div tag of "pad" class will have beige color and background.
If the user strts to minimize the window and the width changes to some width between 500px and 699px, the BG color changes to cadetblue. If the window is less than or equal to 500px, the background color changes to aqua.
The h3 Tags are used as subheadings. so all h3 tags are with cadetblue background
