About the Experiment

The Experiment is about responsive container. The css of the webpage uses the "@media" tag which refers to the screen of the display device. If the display device has a width grater then 700px, the ".pad" class of the body will have a fixed width of text in it, i.e 600px.If the minimizing of the window reaches its next break point, that is 500px - 700px, the text will now be confined to a size of 400px. This is to make sure that the contaoner of the body will always be responsive.
