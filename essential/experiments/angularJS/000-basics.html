About the Experiment

The experiment begins the idea of working with AngularJS. The experiment introduces the two-way binding channels of AngularJS. The directive ng-app initiates the app as a angular application. The angular.module in the script instantiates the angular app. The literals that look like {{hello}} are the two way binders. Whatever happens to them in the UI is reflected in script and viceversa. These are bound by ng-model directive.

Code Snippets

var employee = { first: "Saradhi", last: "Kethamakka" };
$scope.employee = employee;

The above code says that The JSON "employee" has "first" and "last" attributes. These attributes are bound to {{employee.first}} and {{employee.last}}
