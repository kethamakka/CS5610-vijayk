About the Experiment

This experiment allows the user to add a movie that he/she likes to the favorite list of movies. After being added to favorites, The movie can be removed from the list.

$scope.favorites = [];
$scope.addToFavorites = function (movie) {
$scope.favorites.push(movie);
};

The code shows how a movie is being added to the favorites list.
