Description

This experiment is about two components. Firstly the "Pagination". This option gives an index to navigate between different pages from a single webpaage. The Index is diaplayed as a horizontal array of links. Each page, which is a part of "pagination" must have the code for pagination.
The pagination is achieved by assigning class="pagination" to "UL" tag. The following code snippet shows how the pagination is defined

Code Snippet

<ul class="pagination">
<li><a href="004-pagination.html">Home</a></li>
<li class="active"><a href="004-about.html">About</a></li>
<li><a href="004-html.html">Html Source</a></li>
<li><a href="004-css.html">Css Source</a></li>
</ul>

Secondly, in the Home page of the experiment, a "Bootstrap Carousel" is implemented. A Carousel is essentially a Slide show within a webpage. This is a plugin, which is included in "bootstrap.js" file. The Carousel is contained in a "div" tag, which must have an ID, for the carousel functions to perform correctly.
The caoursel is defined by the class "carousl slide". This specifies that the "div" has a carousel. The "data-ride" attribute tells the bootstrap to start the animation as soon as the webpage loads. Tha verious parts of the code in the following code snippets, describe various functions of the carousel.


Code Snippet

<ol class="carousel-indicators">
<li data-target="#myCarousel" data-slide-to="0" class="active"></li>
<li data-target="#myCarousel" data-slide-to="1"></li>
<li data-target="#myCarousel" data-slide-to="2"></li>
<li data-target="#myCarousel" data-slide-to="3"></li>
</ol>

The above code snippet tells bootstrap abou the indicators (small dots on the carousel showing the number of slides in the crousel) as such, which indicator should direct to what slide.


Code Snippet

<div class="item active">
<img class="img-responsive" src="Campus.jpg" alt="My Campus" width="704" height="342">
</div>

The above code snippet shows an example of assigning a file to the slide show of the carousel.


Code Snippet

<a class="left carousel-control" href="#myCarousel" role="button" data-slide="prev">
<span class="glyphicon glyphicon-chevron-left" aria-hidden="true"></span>
<span class="sr-only">Previous</span>
</a>

The above code snippet shows how the arrows on the carousel should work. The carousel provides navigation of slides through "right" and "left" arrows.
