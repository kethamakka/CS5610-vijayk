About Responsive Menu Experiment

The experiment is to demonstrate how a menu on a page could be responsive. Generally menus are displayed as a navigation bar. When the window has to be displayed on a smaller screen , the menu wraps up like text. To avoid this, the menus can be designed responsive.

Code Snippets

#menu-link, #small-menu, #back-to-top{
display:none;
}
@media screen and (max-width:399px){
#menu-link, #small-menu, #back-to-top {
display:inline;
}
these code snippets decide when the content in the menu has to be displayed and when not to. The default behaviour is to keep the menus hidden. But when the screen goes smaller than 400px, navigation bar is hidden and a link to the menu items is shown
