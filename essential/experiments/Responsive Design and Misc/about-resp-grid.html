About the Experiment

The Experiment is about, how, if the page has been arranged as a grid, will the webpage respond to the variations in the scren. the code snippets show how it has been coded

Code Snippet

@media screen and (min-width:400px) and (max-width: 600px){ .pad{ width:375px; }
.row{ width:375px; clear:both; }
.col-6{ width:170px; float:right; }
}
The code says that when the screen is between 400px to 600px, the maximum width should be 375px and it has to be divided amongst two columns each of width 170px.
