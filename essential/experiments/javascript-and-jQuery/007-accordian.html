JQuery Accordion

Description
This experiment is about using JQuery UI Accordian Plugin for creating accordian with some styling applied. For each content under accordion we create a div and include the heading in header tag and content in div tag. The main purpose of this experiment is to make efficient use of space and accordion helps us to use a limited amount of space to show more content. This is achieved by showing the content of class which is currently active and hiding the rest.
