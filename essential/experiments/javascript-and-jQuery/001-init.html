About the Experiment

jQuery(function () {
console.log("The DOM is loaded");
jQuery("h1").remove();
});

In this Experiment, the javascript file invokes a function "jQuery". A function "init" is used as an argument to the jQuery function. This makes the browser delay the execution of "javascript" file until the document is downloaded. Since the Document Object Model will be prepared before the execution of script elements, the script can find H1 elements and will remove it.
